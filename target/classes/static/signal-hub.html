Webhook Setup

Run ngrok to get your public URL (see Step 1 below)
After setting, add ?secret=YOUR_KEY to the webhook URL in TrendSpider
1
Start ngrok to get a public URL
Run this in a terminal (Spring Boot must be running on port 8080 first): 
ngrok http 8080
Copy the https://xxxx.ngrok-free.app URL it shows. Paste it into the Webhook URL box above.
2
Set your security key
Generate a key above, then open application.properties and set: 
trendspider.webhook.secret=YOUR_KEY_HERE
Restart Spring Boot after changing it.
3
Open TrendSpider → Create Alert
Go to your Gold/Silver/Oil/US100 chart → Click the Alert button (bell icon) → Set your TA condition → scroll to Webhook tab.
4
Paste into TrendSpider Webhook tab
URL field:
https://YOUR-NGROK-URL/api/webhook/trendspider?secret=YOUR_KEY
Body field (copy exactly):
{
  "ticker":    "{{ticker}}",
  "price":     {{close}},
  "action":    "BUY",
  "alertName": "{{alert_name}}",
  "interval":  "{{interval}}",
  "message":   "{{alert_message}}",
  "timestamp": "{{time}}"
}
For a SELL signal, create a second alert with "action": "SELL".
5
Test it
Click Test Webhook in TrendSpider. The alert will appear in the Live Feed below instantly.
